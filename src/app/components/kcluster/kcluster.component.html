Standard

Standard clusters will use images created and provided by kubevirt-manager.io initiative.

Cluster also supports different CNI configurations provided by kubevirt-manager.io as well as features like Ingress, Dashboard and Tekton.


Custom

Custom clusters require you to provide the disk images to be imported by the cluster.

CNI needs to be installed manually and features like Ingress, Dashboard and Tekton are not provided.
